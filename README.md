# Minecraft Loader System

Система лоадера для Minecraft с авторизацией по ключам и интеграцией с Discord.

## Компоненты системы

1. **Серверная часть (backend)**
   - API для управления ключами
   - База данных
   - Система аутентификации
   - Интеграция с Discord

2. **Веб-интерфейс (frontend)**
   - Управление ключами
   - Статистика пользователей
   - Система регистрации по инвайтам

3. **Discord бот**
   - Управление ролями
   - Генерация ключей
   - Привязка аккаунтов

4. **Лоадер (client)**
   - Авторизация по ключу
   - Загрузка и запуск модов
   - Безопасное удаление файлов

## Требования к системе

### Сервер (Ubuntu 20.04)
- Python 3.8+
- Node.js 16+
- PostgreSQL 12+
- Nginx
- Redis

### Клиент (Windows)
- Windows 10/11
- .NET Framework 4.7.2+
- Minecraft Java Edition

## Установка

### Серверная часть

1. Установка зависимостей:
```bash
sudo apt update
sudo apt install python3.8 python3.8-venv nodejs npm postgresql nginx redis-server
```

2. Настройка базы данных:
```bash
sudo -u postgres psql
CREATE DATABASE minecraft_loader;
CREATE USER loader_user WITH PASSWORD 'your_password';
GRANT ALL PRIVILEGES ON DATABASE minecraft_loader TO loader_user;
```

3. Клонирование репозитория:
```bash
git clone [repository_url]
cd minecraft-loader
```

4. Установка Python зависимостей:
```bash
python3.8 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

5. Установка Node.js зависимостей:
```bash
cd frontend
npm install
```

6. Настройка переменных окружения:
```bash
cp .env.example .env
# Отредактируйте .env файл, указав необходимые параметры
```

7. Запуск сервисов:
```bash
# Запуск backend
python manage.py runserver

# Запуск frontend
cd frontend
npm run build
npm start

# Запуск Discord бота
python bot.py
```

### Клиентская часть

1. Скачайте последнюю версию лоадера с сайта
2. Запустите установщик
3. Следуйте инструкциям установщика

## Безопасность

- Все ключи шифруются в базе данных
- Используется HTTPS для всех соединений
- Реализована защита от повторной регистрации
- Система автоматического удаления файлов при закрытии лоадера

## Команды Discord бота

### Команды для администраторов
- `/genkey [is_test]` - Генерация нового ключа (опционально: указать тестовый ключ)
- `/ban [@user] [причина]` - Бан пользователя с указанием причины
- `/unban [@user]` - Разбан пользователя

### Команды для пользователей
- `/code` - Получение кода для привязки Discord аккаунта
- `/redeem [ключ]` - Активация ключа
- `/stats` - Просмотр статистики пользователя (количество ключей, приглашенных пользователей и т.д.)

### Команды для поддержки
- `/userinfo [@user]` - Просмотр информации о пользователе (ID, email, дата регистрации и т.д.)
- `/genkey [is_test]` - Генерация нового ключа (действия логируются и отправляются администратору)
- `/ban [@user] [причина]` - Бан пользователя (действия логируются и отправляются администратору)
- `/unban [@user]` - Разбан пользователя (действия логируются и отправляются администратору)

> **Примечание:** Все действия команды поддержки автоматически логируются и отправляются администраторам для контроля.
