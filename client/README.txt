Minecraft Loader
==============

Описание
--------
Minecraft Loader - это программа для запуска Minecraft с системой авторизации по ключам. 
Программа обеспечивает безопасный доступ к игре и автоматически удаляет все временные файлы при закрытии.

Требования
----------
- Windows 10/11
- .NET Framework 4.7.2 или выше
- Minecraft Java Edition
- Java Runtime Environment

Установка
---------
1. Запустите установщик
2. Следуйте инструкциям установщика
3. После установки запустите лоадер
4. Введите ключ для активации

Использование
------------
1. Запустите Minecraft Loader
2. Введите ваш ключ в поле ввода
3. Нажмите кнопку "Запустить"
4. Дождитесь загрузки и запуска игры

Безопасность
-----------
- Все ключи шифруются
- Используется защищенное соединение
- Автоматическое удаление временных файлов
- Защита от повторной регистрации
